# Firebase Studio

Este é um projeto Next.js criado no Firebase Studio.

## Para Começar

Para rodar o ambiente de desenvolvimento, execute:

```bash
npm run dev
```

Isso iniciará o servidor de desenvolvimento em `http://localhost:9002`.

## Para Produção

Para fazer o build da aplicação para produção, execute:

```bash
npm run build
```

Após o build, você pode iniciar o servidor de produção com:

```bash
npm run start
```
