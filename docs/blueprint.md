# **App Name**: Agenda Alego

## Core Features:

- Event Display: Display events in a list format, showing the date, event name, and location.
- Administrative Dashboard: Allows administrators to create events with a date, name, location, and options for 'YouTube' or 'TV Aberta' selection.
- Randomized Event Card Colors: Show different background colors for each event card to enhance visual appeal. Implement random color generation for these event cards.
- Transmission Icons: Integrate Font Awesome icons to visually represent the transmission options (YouTube, TV Aberta) for each event.
- Responsive Design: Ensure the application interface is responsive, adapting to different screen sizes for optimal viewing on various devices.

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC) to convey reliability and organization.
- Background color: Light gray (#F0F4F7) to ensure readability and focus on event information.
- Accent color: Light orange (#F9C9A6) for highlighting interactive elements and call-to-actions.
- Body and headline font: 'PT Sans' for clear and accessible information display.
- Font Awesome icons will be used to indicate the nature of the transmission (YouTube, TV Aberta).
- The app layout is a single list of cards showing event data; it is responsively designed for different screen sizes.
